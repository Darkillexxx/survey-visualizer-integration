# Trivia Visualization App

This is a small React project that shows data from the Open Trivia DB API.  
The app loads 50 random questions and lets you see how they are distributed by category and difficulty.

You can filter the data by category to check how the difficulty changes.  
Everything updates instantly, and the charts are made with Recharts.

### Features
- Shows number of questions by category (bar chart)
- Shows difficulty levels (pie chart)
- Filter by category
- Responsive layout with a clean design

### Stack
React, Recharts, JavaScript, CSS, GitHub Pages
